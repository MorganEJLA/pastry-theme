# Universo da Doçura — pastry-theme

Custom WordPress theme for a fictional international pastry brand, 
built around a love of Brazilian culture and global dessert traditions.

🌐 [Live Site](https://universo-da-docura.com)

---

## Features

- Custom post types: `pastry_case`, `professor`, `locale`
- Advanced Custom Fields (ACF) for rich content modeling
- Swiper.js-powered pastry showcase with custom navigation
- SCSS compiled via `@wordpress/scripts` / wp-scripts
- Custom PHP templates for archives, singles, and page-specific layouts
- MU-plugins for custom functionality
- WCAG 2AA accessibility audit completed via Pa11y

## Tech Stack

- WordPress (classic theme)
- PHP
- SCSS → compiled CSS via wp-scripts
- JavaScript / Swiper.js
- ACF

## Local Development

Requires [LocalWP](https://localwp.com/).

1. Clone into `wp-content/themes/pastry-theme`
2. Run `npm install`
3. Run `npm run start` to watch SCSS
